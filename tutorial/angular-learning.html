Angular - Complete Guide

What is Angular?

Angular is a modern TypeScript-based framework for building single-page applications (SPAs) with a component-based architecture. It provides a complete solution for building dynamic, interactive web applications with built-in dependency injection, routing, forms handling, and HTTP services.

Angular Evolution

  • Angular 2: Complete rewrite in TypeScript (2016)
  • Angular 4-9: Incremental improvements, RxJS integration
  • Angular 12: TypeScript 4.2+ support
  • Angular 14: Typed Forms, standalone components preview
  • Angular 15: Standalone components stable, directive composition
  • Angular 16: Control flow syntax (@if, @for, @switch)
  • Angular 17: New @if/@for/@switch, signals API preview

Angular Architecture - Core Building Blocks

  • Components: Basic UI building blocks with template, class, metadata
  • Modules (NgModule): Logical grouping of components, services, pipes
  • Services: Reusable logic injected via dependency injection
  • Directives: Structural (*ngIf, *ngFor) and attribute directives
  • Pipes: Transform data in templates (date, currency, uppercase)
  • Decorators: @Component, @Injectable, @Input, @Output metadata
  • Dependency Injection: Invert control pattern for service management

Creating a Component (Modern Angular 17)


// Standalone component (Angular 14+)
import { Component, Input, Output, EventEmitter } from '@angular/core';
import { CommonModule } from '@angular/common';

@Component({
  selector: 'app-user-card',
  standalone: true,
  imports: [CommonModule],
  template: `
    

      
{{ user.name }}

      
{{ user.email }}

      
    

  `,
  styles: [`
    .user-card {
      border: 1px solid #ddd;
      padding: 15px;
      border-radius: 8px;
    }
  `]
})
export class UserCardComponent {
  @Input() user: any;
  @Output() delete = new EventEmitter();
  
  onDelete() {
    this.delete.emit(this.user.id);
  }
}

Dependency Injection


// Service definition
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'  // Available app-wide
})
export class UserService {
  constructor(private http: HttpClient) {}
  
  getUsers(): Observable {
    return this.http.get('/api/users');
  }
  
  getUser(id: number): Observable {
    return this.http.get(`/api/users/${id}`);
  }
}

// Using service in component
@Component({
  selector: 'app-users',
  template: '...'
})
export class UsersComponent implements OnInit {
  users$ = new Observable();
  
  // Inject service through constructor
  constructor(private userService: UserService) {}
  
  ngOnInit() {
    this.users$ = this.userService.getUsers();
  }
}

Data Binding & Template Syntax


// Property binding: [] - one way, component to template
<img [src]="imageUrl" [alt]="imageName" />

// Event binding: () - template to component
<button (click)="onClickButton()">Click me</button>
<input (keyup)="onKeyUp($event)" />

// Two-way binding: [()] - both directions
<input [(ngModel)]="userName" />

// Interpolation: {{ }} - template expression
<h1>{{ title }}</h1>
<p>{{ user.firstName + ' ' + user.lastName }}</p>

// Property binding shorthand
<div [title]="description">...</div>

// Binding to custom components
<app-user-card [user]="currentUser" (delete)="onDelete($event)"></app-user-card>

Control Flow (Angular 17+)


// @if - replaces *ngIf
<div *ngIf="isLoggedIn; else notLoggedIn">
  Welcome {{ userName }}
</div>
<ng-template #notLoggedIn>
  Please login
</ng-template>

// Modern syntax (Angular 17)
@if (isLoggedIn) {
  <div>Welcome {{ userName }}</div>
} @else {
  <div>Please login</div>
}

// @for - replaces *ngFor
@for (item of items; track item.id) {
  <li>{{ item.name }} - ${{ item.price }}</li>
}

// @switch - replaces [ngSwitch]
@switch (userRole) {
  @case ('admin') {
    <p>Admin Panel</p>
  }
  @case ('user') {
    <p>User Dashboard</p>
  }
  @default {
    <p>Guest View</p>
  }
}

RxJS Observables & Reactive Programming


import { Observable, Subject, BehaviorSubject } from 'rxjs';
import { map, filter, switchMap, takeUntil } from 'rxjs/operators';

// Observable - one-way stream
const userClick$ = fromEvent(button, 'click');

// Subject - multicasting (many subscribers)
private messageSubject = new Subject();
public message$ = this.messageSubject.asObservable();

// BehaviorSubject - has current value
private userSubject = new BehaviorSubject(defaultUser);
public user$ = this.userSubject.asObservable();

getUser(id: number) {
  return this.http.get(`/api/users/${id}`).pipe(
    map(user => ({ ...user, name: user.name.toUpperCase() })),
    filter(user => user.active),
    switchMap(user => this.getDetails(user.id))
  );
}

// Async pipe auto-unsubscribes
<div>{{ user$ | async | json }}</div>

Reactive Forms (Type-Safe)


import { FormBuilder, FormGroup, Validators, ReactiveFormsModule } from '@angular/forms';

@Component({
  selector: 'app-user-form',
  imports: [ReactiveFormsModule],
  template: `
    <form [formGroup]="form" (ngSubmit)="onSubmit()">
      <input formControlName="name" required />
      <input formControlName="email" type="email" />
      
      <div formGroupName="address">
        <input formControlName="street" />
        <input formControlName="city" />
      </div>
      
      <div *ngIf="form.get('email')?.hasError('email')">
        Invalid email
      </div>
      
      <button [disabled]="form.invalid">Submit</button>
    </form>
  `
})
export class UserFormComponent {
  form: FormGroup;
  
  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      name: ['', [Validators.required]],
      email: ['', [Validators.required, Validators.email]],
      address: this.fb.group({
        street: [''],
        city: ['']
      })
    });
  }
  
  onSubmit() {
    if (this.form.valid) {
      console.log(this.form.value);
    }
  }
}

HTTP Services with Error Handling


import { HttpClient, HttpErrorResponse } from '@angular/common/http';
import { catchError, retry } from 'rxjs/operators';
import { throwError } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  constructor(private http: HttpClient) {}
  
  getUsers() {
    return this.http.get('/api/users').pipe(
      retry(1),
      catchError(this.handleError)
    );
  }
  
  createUser(user: any) {
    return this.http.post('/api/users', user).pipe(
      catchError(error => {
        console.error('Error creating user:', error);
        return throwError(() => new Error('Failed to create user'));
      })
    );
  }
  
  private handleError(error: HttpErrorResponse) {
    let errorMsg = 'Unknown error occurred';
    
    if (error.error instanceof ErrorEvent) {
      errorMsg = `Client Error: ${error.error.message}`;
    } else {
      errorMsg = `Server Error: ${error.status} - ${error.message}`;
    }
    
    return throwError(() => new Error(errorMsg));
  }
}

Custom Directives


import { Directive, ElementRef, HostListener, Input } from '@angular/core';

// Attribute Directive - modify behavior
@Directive({
  selector: '[appHighlight]',
  standalone: true
})
export class HighlightDirective {
  @Input() appHighlight = 'yellow';
  
  constructor(private el: ElementRef) {}
  
  @HostListener('mouseenter')
  onMouseEnter() {
    this.el.nativeElement.style.backgroundColor = this.appHighlight;
  }
  
  @HostListener('mouseleave')
  onMouseLeave() {
    this.el.nativeElement.style.backgroundColor = null;
  }
}

// Usage: <div appHighlight="lightblue">Hover me</div>

Pipes (Data Transformation)


// Built-in pipes
{{ date | date:'short' }}
{{ price | currency:'USD' }}
{{ text | uppercase }}
{{ items | slice:0:3 }}
{{ obj | json }}
{{ 0.259 | percent }}

// Custom pipe
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'titleCase',
  standalone: true
})
export class TitleCasePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value
      .toLowerCase()
      .split(' ')
      .map(word => word.charAt(0).toUpperCase() + word.slice(1))
      .join(' ');
  }
}

// Usage: {{ title | titleCase }}

State Management with Signals (Angular 17)


import { signal, computed, effect } from '@angular/core';

export class CounterComponent {
  // Create reactive signal
  count = signal(0);
  
  // Computed signal - auto updates when dependency changes
  doubled = computed(() => this.count() * 2);
  
  // Effect - runs when signal changes
  constructor() {
    effect(() => {
      console.log(`Count changed to: ${this.count()}`);
    });
  }
  
  increment() {
    this.count.update(val => val + 1);
  }
  
  reset() {
    this.count.set(0);
  }
}

// In template
<p>Count: {{ count() }}</p>
<p>Doubled: {{ doubled() }}</p>
<button (click)="increment()">Increment</button>

Common Interview Problems & Patterns

  • Change Detection: OnPush vs Default strategy
  • Memory Leaks: Unsubscribe patterns, takeUntil operator

Best Practices

  • Smart vs Presentational Components: Separate data/logic from UI
  • OnPush Change Detection: Better performance with immutable patterns
  • Unsubscribe Pattern: Prevent memory leaks with takeUntil()
  • Standalone Components: Modern approach (Angular 14+)
  • Async Pipe: Auto-subscribe/unsubscribe in templates
  • Type Safety: Use typed forms and strict TypeScript
  • Lazy Loading: Split code by routes for faster initial load
  • Code Organization: Feature modules, shared module pattern
