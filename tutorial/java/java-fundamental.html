üöÄ Java Overview

What is Java?

Definition: Java is a high-level, object-oriented, platform-independent programming language designed for building robust, secure, and portable applications.

Key Features of Java:

  • Platform Independent: "Write Once, Run Anywhere" (WORA)
  • Object-Oriented: Everything is an object (except primitives)
  • Robust: Strong memory management and exception handling
  • Secure: Built-in security features and sandboxing
  • Multithreaded: Built-in support for concurrent programming
  • Automatic Memory Management: Garbage collection

Java Architecture:

JVM, JRE, and JDK:

  • JVM (Java Virtual Machine): Executes Java bytecode
  • JRE (Java Runtime Environment): JVM + Libraries needed to run Java programs
  • JDK (Java Development Kit): JRE + Development tools (compiler, debugger)

        // String literal pool
        String str3 = "Hello";
        String str4 = "Hello";
        System.out.println("String literal comparison (str3 == str4): " + (str3 == str4)); // true
    }

Type Conversion:

public class TypeConversionExample {
    
    public static void main(String[] args) {
        // Implicit conversion (widening)
        byte byteVal = 10;
        short shortVal = byteVal;    // byte to short
        int intVal = shortVal;       // short to int
        long longVal = intVal;       // int to long
        float floatVal = longVal;    // long to float
        double doubleVal = floatVal; // float to double
        
        System.out.println("=== Implicit Conversion ===");
        System.out.println("byte to double: " + byteVal + " -> " + doubleVal);
        
        // Explicit conversion (narrowing) - casting required
        double d = 123.456;
        float f = (float) d;         // double to float
        long l = (long) f;           // float to long
        int i = (int) l;             // long to int
        short s = (short) i;         // int to short
        byte b = (byte) s;           // short to byte
        
        System.out.println("\n=== Explicit Conversion ===");
        System.out.println("double to byte: " + d + " -> " + b);
        
        // String conversions
        demonstrateStringConversions();
        
        // Wrapper class conversions
        demonstrateWrapperConversions();
    }
    
    public static void demonstrateStringConversions() {
        System.out.println("\n=== String Conversions ===");
        
        // Primitive to String
        int number = 42;
        String strFromInt = String.valueOf(number);
        String strFromInt2 = Integer.toString(number);
        String strFromInt3 = "" + number; // Concatenation
        
        System.out.println("int to String: " + strFromInt);
        
        // String to primitive
        String numberStr = "123";
        int intFromStr = Integer.parseInt(numberStr);
        double doubleFromStr = Double.parseDouble("123.45");
        boolean boolFromStr = Boolean.parseBoolean("true");
        
        System.out.println("String to int: " + intFromStr);
        System.out.println("String to double: " + doubleFromStr);
        System.out.println("String to boolean: " + boolFromStr);
    }
    
    public static void demonstrateWrapperConversions() {
        System.out.println("\n=== Wrapper Class Conversions ===");
        
        // Autoboxing - primitive to wrapper
        Integer integerObj = 100;        // int to Integer
        Double doubleObj = 3.14;         // double to Double
        Boolean booleanObj = true;       // boolean to Boolean
        
        // Unboxing - wrapper to primitive
        int primitiveInt = integerObj;   // Integer to int
        double primitiveDouble = doubleObj; // Double to double
        boolean primitiveBool = booleanObj; // Boolean to boolean
        
        System.out.println("Autoboxing: " + integerObj.getClass().getSimpleName());
        System.out.println("Unboxing: " + primitiveInt);
        
        // Null pointer risk with unboxing
        Integer nullInteger = null;
        try {
            int risk = nullInteger; // NullPointerException
        } catch (NullPointerException e) {
            System.out.println("Unboxing null wrapper causes: " + e.getClass().getSimpleName());
        }
    }
}

Types of Variables

public class VariableTypesExample {
    
    // 1. Instance variables (non-static fields)
    private String instanceName;
    private int instanceAge;
    
    // 2. Class variables (static fields)
    private static int totalObjects = 0;
    public static final String COMPANY_NAME = "TechCorp";
    
    // Constructor
    public VariableTypesExample(String name, int age) {
        // 3. Local variables
        String message = "Creating new object"; // Local variable
        System.out.println(message);
        
        // Initialize instance variables
        this.instanceName = name;
        this.instanceAge = age;
        
        // Update class variable
        totalObjects++;
    }
    
    public void demonstrateVariables() {
        // 4. Parameter variables
        processData("Sample Data", 42);
        
        // Local variables in method
        int localCounter = 0;
        String localMessage = "Method execution";
        
        // Block variables
        if (instanceAge > 18) {
            String adultMessage = "Adult user"; // Block scope
            System.out.println(adultMessage);
        }
        
        // Loop variables
        for (int i = 0; i < 3; i++) { // i is loop variable
            System.out.println("Loop iteration: " + i);
        }
        
        System.out.println("Local counter: " + localCounter);
        System.out.println("Instance name: " + instanceName);
        System.out.println("Total objects: " + totalObjects);
    }
    
    // Method with parameters
    public void processData(String data, int count) {
        // data and count are parameter variables
        System.out.println("Processing: " + data + " (count: " + count + ")");
        
        // Parameters can be modified (but it's local copy)
        data = data.toUpperCase();
        count = count * 2;
        System.out.println("Modified locally: " + data + " (count: " + count + ")");
    }
    
    // Static method can only access static variables directly
    public static void displayStaticInfo() {
        System.out.println("Company: " + COMPANY_NAME);
        System.out.println("Total objects created: " + totalObjects);
        
        // Cannot access instance variables directly
        // System.out.println(instanceName); // Compilation error
    }
    
    public static void main(String[] args) {
        System.out.println("=== Variable Types Demo ===");
        
        VariableTypesExample obj1 = new VariableTypesExample("Alice", 25);
        VariableTypesExample obj2 = new VariableTypesExample("Bob", 30);
        
        obj1.demonstrateVariables();
        obj2.demonstrateVariables();
        
        displayStaticInfo();
    }
}

Variable Modifiers:

public class VariableModifiersExample {
    
    // Access modifiers
    public int publicVar = 1;        // Accessible from anywhere
    protected int protectedVar = 2;  // Accessible within package and subclasses
    int packageVar = 3;              // Accessible within package (default)
    private int privateVar = 4;      // Accessible only within this class
    
    // Non-access modifiers
    static int staticVar = 5;        // Belongs to class, not instance
    final int finalVar = 6;          // Cannot be changed after initialization
    volatile int volatileVar = 7;    // Thread-safe access
    transient int transientVar = 8;  // Not serialized
    
    // Combinations
    public static final String CONSTANT = "IMMUTABLE";

        instanceCount++;

        System.out.println("Package: " + packageVar);
        System.out.println("Private: " + privateVar);
        System.out.println("Static: " + staticVar);
        System.out.println("Final: " + finalVar);
        System.out.println("Constant: " + CONSTANT);

        // finalVar = 10; // Compilation error
        
        // Static variables can be accessed via class name

Arithmetic Operators

public class ArithmeticOperatorsExample {
    
    public static void main(String[] args) {
        int a = 15, b = 4;
        
        System.out.println("=== Arithmetic Operators ===");
        System.out.println("a = " + a + ", b = " + b);
        
        // Basic arithmetic
        System.out.println("Addition (a + b): " + (a + b));        // 19
        System.out.println("Subtraction (a - b): " + (a - b));     // 11
        System.out.println("Multiplication (a * b): " + (a * b));  // 60
        System.out.println("Division (a / b): " + (a / b));        // 3 (integer division)
        System.out.println("Modulus (a % b): " + (a % b));         // 3 (remainder)
        
        // Floating-point division
        double result = (double) a / b;
        System.out.println("Floating division: " + result);        // 3.75
        
        // Unary operators
        System.out.println("\n=== Unary Operators ===");
        int x = 10;
        System.out.println("Original x: " + x);
        System.out.println("Unary plus (+x): " + (+x));           // 10
        System.out.println("Unary minus (-x): " + (-x));          // -10
        
        // Increment and decrement
        System.out.println("\n=== Increment/Decrement ===");
        int y = 5;
        System.out.println("Original y: " + y);
        System.out.println("Pre-increment (++y): " + (++y));      // 6
        System.out.println("Post-increment (y++): " + (y++));     // 6, then y becomes 7
        System.out.println("After post-increment y: " + y);       // 7
        System.out.println("Pre-decrement (--y): " + (--y));      // 6
        System.out.println("Post-decrement (y--): " + (y--));     // 6, then y becomes 5
        System.out.println("Final y: " + y);                      // 5
        
        // Compound assignment operators
        System.out.println("\n=== Compound Assignment ===");
        int z = 10;
        System.out.println("Original z: " + z);
        z += 5;  // z = z + 5
        System.out.println("After z += 5: " + z);                 // 15
        z -= 3;  // z = z - 3
        System.out.println("After z -= 3: " + z);                 // 12
        z *= 2;  // z = z * 2
        System.out.println("After z *= 2: " + z);                 // 24
        z /= 4;  // z = z / 4
        System.out.println("After z /= 4: " + z);                 // 6
        z %= 4;  // z = z % 4
        System.out.println("After z %= 4: " + z);                 // 2
    }
}

Comparison and Logical Operators

public class ComparisonLogicalOperatorsExample {
    
    public static void main(String[] args) {
        int a = 10, b = 20, c = 10;
        
        System.out.println("=== Comparison Operators ===");
        System.out.println("a = " + a + ", b = " + b + ", c = " + c);
        
        // Comparison operators
        System.out.println("a == b: " + (a == b));                // false
        System.out.println("a == c: " + (a == c));                // true
        System.out.println("a != b: " + (a != b));                // true
        System.out.println("a < b: " + (a < b));                  // true
        System.out.println("a > b: " + (a > b));                  // false
        System.out.println("a <= c: " + (a <= c));                // true
        System.out.println("a >= c: " + (a >= c));                // true
        
        // Logical operators
        System.out.println("\n=== Logical Operators ===");
        boolean x = true, y = false;
        System.out.println("x = " + x + ", y = " + y);
        
        System.out.println("x && y (AND): " + (x && y));          // false
        System.out.println("x || y (OR): " + (x || y));           // true
        System.out.println("!x (NOT): " + (!x));                  // false
        System.out.println("!y (NOT): " + (!y));                  // true
        
        // Short-circuit evaluation
        System.out.println("\n=== Short-Circuit Evaluation ===");
        int m = 0, n = 5;
        
        // && short-circuit: if first is false, second is not evaluated
        boolean result1 = (m != 0) && (n / m > 2);
        System.out.println("Short-circuit AND result: " + result1); // false, no division by zero
        
        // || short-circuit: if first is true, second is not evaluated
        boolean result2 = (n > 0) || (n / m > 2);
        System.out.println("Short-circuit OR result: " + result2);  // true, no division by zero
        
        // Complex logical expressions
        System.out.println("\n=== Complex Expressions ===");
        int age = 25;
        boolean hasLicense = true;
        boolean hasInsurance = false;
        
        boolean canDrive = (age >= 18) && hasLicense && hasInsurance;
        System.out.println("Can drive: " + canDrive);              // false
        
        boolean isEligible = (age >= 21 && age <= 65) || (hasLicense && age >= 18);
        System.out.println("Is eligible: " + isEligible);          // true
    }
}

Bitwise and Other Operators

public class BitwiseOtherOperatorsExample {
    
    public static void main(String[] args) {
        // Bitwise operators
        System.out.println("=== Bitwise Operators ===");
        int a = 12;  // Binary: 1100
        int b = 10;  // Binary: 1010
        
        System.out.println("a = " + a + " (binary: " + Integer.toBinaryString(a) + ")");
        System.out.println("b = " + b + " (binary: " + Integer.toBinaryString(b) + ")");
        
        System.out.println("a & b (AND): " + (a & b) + " (binary: " + Integer.toBinaryString(a & b) + ")");   // 8 (1000)
        System.out.println("a | b (OR): " + (a | b) + " (binary: " + Integer.toBinaryString(a | b) + ")");    // 14 (1110)
        System.out.println("a ^ b (XOR): " + (a ^ b) + " (binary: " + Integer.toBinaryString(a ^ b) + ")");   // 6 (0110)
        System.out.println("~a (NOT): " + (~a) + " (binary: " + Integer.toBinaryString(~a) + ")");            // -13
        
        // Shift operators
        System.out.println("\n=== Shift Operators ===");
        int x = 8;  // Binary: 1000
        System.out.println("x = " + x + " (binary: " + Integer.toBinaryString(x) + ")");
        
        System.out.println("x << 2 (Left shift): " + (x << 2) + " (binary: " + Integer.toBinaryString(x << 2) + ")");   // 32 (100000)
        System.out.println("x >> 2 (Right shift): " + (x >> 2) + " (binary: " + Integer.toBinaryString(x >> 2) + ")");  // 2 (10)
        
        int negative = -8;
        System.out.println("negative >> 2: " + (negative >> 2));     // -2 (sign-extended)
        System.out.println("negative >>> 2: " + (negative >>> 2));   // Large positive (zero-filled)
        
        // Ternary operator (conditional operator)
        System.out.println("\n=== Ternary Operator ===");
        int age = 20;
        String status = (age >= 18) ? "Adult" : "Minor";
        System.out.println("Age: " + age + ", Status: " + status);
        
        int max = (a > b) ? a : b;
        System.out.println("Max of " + a + " and " + b + ": " + max);
        
        // Nested ternary
        int score = 85;
        String grade = (score >= 90) ? "A" : (score >= 80) ? "B" : (score >= 70) ? "C" : "F";
        System.out.println("Score: " + score + ", Grade: " + grade);
        
        // instanceof operator
        System.out.println("\n=== instanceof Operator ===");
        String str = "Hello";
        Object obj = str;
        
        System.out.println("str instanceof String: " + (str instanceof String));     // true
        System.out.println("obj instanceof String: " + (obj instanceof String));     // true
        System.out.println("str instanceof Object: " + (str instanceof Object));     // true
        System.out.println("null instanceof String: " + (null instanceof String));   // false
        
        // Operator precedence example
        System.out.println("\n=== Operator Precedence ===");
        int result1 = 2 + 3 * 4;           // 14 (not 20)
        int result2 = (2 + 3) * 4;         // 20
        boolean result3 = true || false && false;  // true (AND has higher precedence)
        boolean result4 = (true || false) && false; // false
        
        System.out.println("2 + 3 * 4 = " + result1);
        System.out.println("(2 + 3) * 4 = " + result2);
        System.out.println("true || false && false = " + result3);
        System.out.println("(true || false) && false = " + result4);
    }
}

üíº Interview Questions

Frequently Asked Questions

Q1: What is the difference between JDK, JRE, and JVM?

Answer:

  • JVM: Java Virtual Machine - executes bytecode, platform-specific
  • JRE: Java Runtime Environment - JVM + libraries to run Java programs
  • JDK: Java Development Kit - JRE + development tools (compiler, debugger)

Q2: What are the differences between primitive and reference data types?

Answer:

  • Primitive: Store actual values, stack memory, 8 types, no methods
  • Reference: Store memory addresses, heap memory, unlimited types, have methods
  • Comparison: Primitives use ==, references use equals()

Q3: Explain autoboxing and unboxing

Answer:

  • Autoboxing: Automatic conversion of primitive to wrapper class
  • Unboxing: Automatic conversion of wrapper class to primitive
  • Risk: NullPointerException when unboxing null wrapper objects

Q4: What is the difference between == and equals()?

Answer:

  • ==: Compares references (memory addresses) for objects, values for primitives
  • equals(): Compares content/values of objects (if properly overridden)
  • String: == compares references, equals() compares content

Q5: What are the different types of variables in Java?

Answer:

  • Instance variables: Non-static fields, belong to object instances
  • Class variables: Static fields, belong to class, shared by all instances
  • Local variables: Declared inside methods, blocks, or constructors
  • Parameter variables: Method parameters, constructor parameters
