What are Pattern Programs? Programs that create visual patterns using characters, numbers, or symbols arranged in specific geometric shapes.

Why Learn Pattern Programming? Develops logical thinking, loop control mastery, and problem-solving skills essential for programming interviews.

How to Approach? Understand row-column relationships, nested loops, and mathematical formulas for spaces and characters.

Star Patterns

1. Right Triangle Star Pattern

public class RightTriangle {
    public static void printRightTriangle(int n) {
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("* ");
            }
            System.out.println();
        }
    }
    
    public static void main(String[] args) {
        System.out.println("Right Triangle Pattern:");
        printRightTriangle(5);
    }
}

* *

9. Pascal's Triangle

public class PascalTriangle {
    public static void printPascalTriangle(int n) {
        for (int i = 0; i < n; i++) {
            // Print spaces for alignment
            for (int j = 0; j < n - i - 1; j++) {
                System.out.print(" ");
            }
            
            int number = 1;
            for (int j = 0; j <= i; j++) {
                System.out.print(number + " ");
                number = number * (i - j) / (j + 1);
            }
            System.out.println();
        }
    }
    
    public static void main(String[] args) {
        System.out.println("Pascal's Triangle:");
        printPascalTriangle(6);
    }
}

/* Output:
     1 
    1 1 
   1 2 1 
  1 3 3 1 
 1 4 6 4 1 
1 5 10 10 5 1 
*/

            // Print spaces
            for (int j = 1; j <= n - i; j++) {

            // Print numbers (ascending)
            for (int j = 1; j <= i; j++) {
                System.out.print(j);

public class AlphabetPyramid {

            char ch = 'A';
            for (int j = 1; j <= i; j++) {

Hollow Patterns

20. Hollow Rectangle

public class HollowRectangle {
    public static void printHollowRectangle(int rows, int cols) {
        for (int i = 1; i <= rows; i++) {
            for (int j = 1; j <= cols; j++) {
                if (i == 1 || i == rows || j == 1 || j == cols) {
                    System.out.print("* ");
                } else {
                    System.out.print("  ");
                }
            }
            System.out.println();
        }
    }
    
    public static void main(String[] args) {
        System.out.println("Hollow Rectangle:");
        printHollowRectangle(5, 8);
    }
}

/* Output:
* * * * * * * * 
*             * 
*             * 
*             * 
* * * * * * * * 
*/

23. Butterfly Pattern

public class ButterflyPattern {
    public static void printButterflyPattern(int n) {
        // Upper half
        for (int i = 1; i <= n; i++) {
            // Left stars
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            // Spaces
            for (int j = 1; j <= 2 * (n - i); j++) {
                System.out.print(" ");
            }
            // Right stars
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
        
        // Lower half
        for (int i = n; i >= 1; i--) {
            // Left stars
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            // Spaces
            for (int j = 1; j <= 2 * (n - i); j++) {
                System.out.print(" ");
            }
            // Right stars
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
    
    public static void main(String[] args) {
        System.out.println("Butterfly Pattern:");
        printButterflyPattern(4);
    }
}

ðŸŽ¯ Pattern Programming Tips

  • Understand the Structure: Analyze rows, columns, spaces, and characters
  • Find the Formula: Identify mathematical relationships between row/column positions
  • Use Nested Loops: Outer loop for rows, inner loops for spaces and characters
  • Practice Visualization: Draw patterns on paper before coding
  • Start Simple: Master basic patterns before attempting complex ones
