üöÄ CSS Overview

What is CSS?

Definition: CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and styling of HTML documents. It controls layout, colors, fonts, and visual appearance.

Key Features:

  • Separation of Concerns: Separates content (HTML) from presentation (CSS)
  • Cascading: Multiple stylesheets can be applied with inheritance rules
  • Responsive: Adapts to different screen sizes and devices
  • Reusable: Styles can be applied to multiple elements
  • Maintainable: Centralized styling makes updates easier

CSS Integration Methods:

/* 1. External CSS (Recommended) */
<link rel="stylesheet" href="styles.css">

/* 2. Internal CSS */
<style>
    body { background-color: #f0f0f0; }
</style>

/* 3. Inline CSS (Avoid when possible) */
<p style="color: red; font-size: 16px;">Styled text</p>

CSS Evolution:

CSS1 (1996) ‚Üí Basic styling properties
CSS2 (1998) ‚Üí Positioning, media types
CSS2.1 (2004) ‚Üí Bug fixes and clarifications
CSS3 (2011+) ‚Üí Modules: Flexbox, Grid, Animations
CSS4 (Future) ‚Üí Advanced selectors and features

Advanced Selectors

/* Combinators */
/* Descendant selector (space) */

üì¶ Box Model

Understanding the Box Model

/* Box Model Components */
.box {
    /* Content area */
    width: 200px;
    height: 100px;
    
    /* Padding (inside the border) */
    padding: 20px;                    /* All sides */
    padding: 10px 20px;               /* Vertical | Horizontal */
    padding: 10px 15px 20px 25px;     /* Top | Right | Bottom | Left */
    
    /* Border */
    border: 2px solid #333;           /* Width | Style | Color */
    border-width: 1px 2px 3px 4px;    /* Individual sides */
    border-style: solid dashed dotted double;
    border-color: red green blue yellow;
    
    /* Margin (outside the border) */
    margin: 10px;                     /* All sides */
    margin: 20px auto;                /* Vertical | Horizontal (auto centers) */
    margin: 10px 15px 20px 25px;      /* Top | Right | Bottom | Left */
}

/* Box-sizing property */
.content-box {
    box-sizing: content-box;  /* Default: width/height = content only */
    width: 200px;             /* Total width = 200px + padding + border */
    padding: 20px;
    border: 5px solid black;  /* Actual total width = 250px */
}

.border-box {
    box-sizing: border-box;   /* width/height includes padding and border */
    width: 200px;             /* Total width = 200px (including padding + border) */
    padding: 20px;
    border: 5px solid black;  /* Content width = 150px */
}

/* Universal box-sizing (recommended) */
*, *::before, *::after {
    box-sizing: border-box;
}

Display Properties

/* Display values */
.block { display: block; }           /* Full width, new line */
.inline { display: inline; }         /* Content width, same line */
.inline-block { display: inline-block; } /* Content width, same line, accepts width/height */
.none { display: none; }             /* Completely hidden */
.flex { display: flex; }             /* Flexible box layout */
.grid { display: grid; }             /* Grid layout */
.table { display: table; }           /* Table-like behavior */

/* Visibility */
.hidden { visibility: hidden; }      /* Hidden but takes space */
.visible { visibility: visible; }    /* Default visible state */

/* Overflow */
.overflow-examples {
    width: 200px;
    height: 100px;
}

.overflow-visible { overflow: visible; }  /* Default: content overflows */
.overflow-hidden { overflow: hidden; }    /* Hide overflowing content */
.overflow-scroll { overflow: scroll; }    /* Always show scrollbars */
.overflow-auto { overflow: auto; }        /* Show scrollbars when needed */

üîÑ Flexbox

Flex Container Properties

/* Flex Container */
.flex-container {
    display: flex;
    
    /* Direction */
    flex-direction: row;        /* Default: left to right */
    flex-direction: row-reverse; /* Right to left */
    flex-direction: column;     /* Top to bottom */
    flex-direction: column-reverse; /* Bottom to top */
    
    /* Wrapping */
    flex-wrap: nowrap;          /* Default: single line */
    flex-wrap: wrap;            /* Multi-line */
    flex-wrap: wrap-reverse;    /* Multi-line, reverse order */
    
    /* Shorthand */
    flex-flow: row wrap;        /* direction + wrap */
    
    /* Main axis alignment */
    justify-content: flex-start;    /* Default: start */
    justify-content: flex-end;      /* End */
    justify-content: center;        /* Center */
    justify-content: space-between; /* Space between items */
    justify-content: space-around;  /* Space around items */
    justify-content: space-evenly;  /* Equal space */
    
    /* Cross axis alignment */
    align-items: stretch;       /* Default: fill container */
    align-items: flex-start;    /* Start of cross axis */
    align-items: flex-end;      /* End of cross axis */
    align-items: center;        /* Center of cross axis */
    align-items: baseline;      /* Baseline alignment */
    
    /* Multi-line alignment */
    align-content: stretch;     /* Default */
    align-content: flex-start;
    align-content: center;
    align-content: space-between;
    
    /* Gap between items */
    gap: 20px;                  /* Row and column gap */
    row-gap: 10px;              /* Row gap only */
    column-gap: 15px;           /* Column gap only */
}

Flex Item Properties

/* Flex Items */
.flex-item {
    /* Growth factor */
    flex-grow: 0;               /* Default: don't grow */
    flex-grow: 1;               /* Grow to fill space */
    flex-grow: 2;               /* Grow twice as much */
    
    /* Shrink factor */
    flex-shrink: 1;             /* Default: can shrink */
    flex-shrink: 0;             /* Don't shrink */
    flex-shrink: 2;             /* Shrink twice as much */
    
    /* Base size */
    flex-basis: auto;           /* Default: content size */
    flex-basis: 200px;          /* Fixed base size */
    flex-basis: 30%;            /* Percentage base size */
    
    /* Shorthand */
    flex: 1;                    /* grow: 1, shrink: 1, basis: 0% */
    flex: 0 1 auto;             /* grow: 0, shrink: 1, basis: auto */
    flex: 200px;                /* grow: 1, shrink: 1, basis: 200px */
    
    /* Individual alignment */
    align-self: auto;           /* Default: inherit from container */
    align-self: flex-start;
    align-self: center;
    align-self: flex-end;
    align-self: stretch;
    
    /* Order */
    order: 0;                   /* Default: source order */
    order: 1;                   /* Move to end */
    order: -1;                  /* Move to beginning */
}

    display: grid;

        "header header header"

    justify-items: start;       /* Align to start */
    justify-items: center;      /* Center items */
    justify-items: end;         /* Align to end */

üíº Interview Questions

Frequently Asked Questions

Q1: What is the CSS Box Model?

Answer: The box model consists of content, padding, border, and margin. Box-sizing property controls whether width/height includes padding and border (border-box) or just content (content-box).

Q2: What's the difference between Flexbox and CSS Grid?

Answer:

  • Flexbox: One-dimensional layout (row or column), content-first
  • Grid: Two-dimensional layout (rows and columns), layout-first
  • Use Flexbox: For component layout, navigation bars
  • Use Grid: For page layout, complex 2D arrangements

Q3: Explain CSS specificity

Answer: Specificity determines which CSS rule applies when multiple rules target the same element. Order: Inline styles (1000) > IDs (100) > Classes/attributes/pseudo-classes (10) > Elements (1).

Q4: What are CSS preprocessors?

Answer: Tools like Sass, Less, and Stylus that extend CSS with variables, nesting, mixins, and functions. They compile to regular CSS for browser compatibility.

Q5: How do you center an element horizontally and vertically?

Answer: Multiple methods: Flexbox (justify-content: center; align-items: center), Grid (place-items: center), or absolute positioning with transform.
