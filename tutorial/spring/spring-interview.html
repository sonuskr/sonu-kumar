üî• TOP SPRING MVC INTERVIEW QUESTIONS (Most Asked)

1. What is Spring MVC?

Spring MVC (Model‚ÄìView‚ÄìController) is a web framework for building web applications & REST APIs.

2. Explain MVC architecture.

Model: Data & business logic
View: UI (HTML, JSP, Thymeleaf)
Controller: Handles requests and returns responses

3. What is DispatcherServlet?

The front controller in Spring MVC.
It receives all incoming requests and dispatches them to appropriate controllers.

4. What is @Controller?

Used to create web controllers that return HTML views.

5. What is @RestController?

Combination of @Controller + @ResponseBody.
Returns JSON instead of HTML.

6. Difference: @Controller vs @RestController

@Controller@RestController
Returns HTML pagesReturns JSON
Used for web appsUsed for REST APIs

7. What is @RequestMapping?

Maps HTTP requests to controller methods.

8. What are @GetMapping, @PostMapping?

Specialized versions of @RequestMapping for specific HTTP methods.

9. What is @PathVariable?

Extracts values from the URL.

@GetMapping("/users/{id}")
public User getUser(@PathVariable int id) { }

10. What is @RequestParam?

Extracts query parameters.

@GetMapping("/search")
public String search(@RequestParam String keyword) { }

11. What is Model?

Used to send data from controller to view.

model.addAttribute("msg", "Hello");

12. What is ModelAndView?

Object that carries both model data and view name.

13. What is ViewResolver?

Resolves logical view names to actual view files.
Example: home ‚Üí home.html or home.jsp

14. What is InternalResourceViewResolver?

Used to resolve JSP files.

15. What is @ResponseBody?

Tells Spring to return JSON instead of a view.

16. What is form validation in Spring MVC?

Use: @Valid and BindingResult

@PostMapping("/save")
public String save(@Valid User user, BindingResult result) { }

17. What is @ModelAttribute?

Binds form data to a model object.

18. What is @InitBinder?

Customizes how data is bound (e.g., date formats).

19. Difference between @RequestBody and @ModelAttribute?

@RequestBody@ModelAttribute
Reads JSONReads form data
Used for REST APIsUsed for MVC forms

20. What is HandlerInterceptor?

Used for pre-processing and post-processing logic (logging, auth).

21. What is Exception Handling in Spring MVC?

Three ways:
‚Ä¢ @ExceptionHandler
‚Ä¢ @ControllerAdvice
‚Ä¢ @RestControllerAdvice

22. How to handle global exceptions?

Using: @RestControllerAdvice

23. What is FlashAttribute?

Used to send data during redirects.

24. What is the role of Web.xml in Spring MVC?

Old style configuration of DispatcherServlet (not required in Spring Boot).

25. Can Spring MVC return JSON?

Yes ‚Üí using @RestController or @ResponseBody.

26. What is Jackson in Spring MVC?

Library used for JSON serialization & deserialization.

27. What is @CrossOrigin?

Enables CORS support for APIs.

28. What is BindingResult?

Captures validation errors when using @Valid.

29. Difference between @RequestBody and @ResponseBody?

@RequestBody ‚Üí Converts JSON ‚Üí Java object
@ResponseBody ‚Üí Converts Java ‚Üí JSON

30. What is ResourceHandler?

Used to expose static resources like images, CSS, JS.

üî• TOP SPRING BOOT INTERVIEW QUESTIONS (Most Asked)

1. What is Spring Boot?

Spring Boot is an extension of Spring that makes development simple with:
‚Ä¢ Auto-configuration
‚Ä¢ Embedded servers
‚Ä¢ Starter dependencies
‚Ä¢ Production-ready features (Actuator)

2. Advantages of Spring Boot

‚Ä¢ No XML configuration
‚Ä¢ Embedded Tomcat/Jetty
‚Ä¢ Faster development
‚Ä¢ Easy integration with Spring modules
‚Ä¢ Microservice-ready
‚Ä¢ Production monitoring via Actuator

3. What are Spring Boot Starters?

Predefined dependencies that simplify build configuration.
Examples:
‚Ä¢ spring-boot-starter-web
‚Ä¢ spring-boot-starter-data-jpa
‚Ä¢ spring-boot-starter-security

4. What is Auto Configuration?

Spring Boot automatically configures beans based on classpath dependencies.
Example: If H2 DB is added, Spring auto-configures DataSource.

5. What is @SpringBootApplication?

It is a meta-annotation that combines:
‚Ä¢ @Configuration
‚Ä¢ @EnableAutoConfiguration
‚Ä¢ @ComponentScan

6. What is Spring Boot Actuator?

Provides production-ready endpoints like:
‚Ä¢ /actuator/health
‚Ä¢ /actuator/metrics
‚Ä¢ /actuator/info
Used for monitoring.

7. How to disable auto-configuration?

8. What is application.properties vs application.yml?

Both are configuration files.
.yml is hierarchical and cleaner for complex configs.

9. What is Embedded Server?

Spring Boot applications run with built-in servers:
‚Ä¢ Tomcat (default)
‚Ä¢ Jetty
‚Ä¢ Undertow
Run as a standalone JAR.

10. How to change server port?

11. Difference between @Component, @Service, @Repository?

‚Ä¢ @Component ‚Üí generic bean
‚Ä¢ @Service ‚Üí business logic
‚Ä¢ @Repository ‚Üí DB layer + exception translation

12. What is @RestController?

Combination of:
‚Ä¢ @Controller
‚Ä¢ @ResponseBody
Used for REST APIs.

13. What is @PathVariable vs @RequestParam?

‚Ä¢ @PathVariable ‚Üí part of URL
‚Ä¢ @RequestParam ‚Üí query parameter

14. How to handle exceptions globally?

Using @RestControllerAdvice.

public class GlobalExceptionHandler { }

15. What is Spring Boot DevTools?

Module for hot reloading and faster development.

16. What is @ConfigurationProperties?

Used to map properties into POJO.

@ConfigurationProperties(prefix="app")

17. How do you secure APIs in Spring Boot?

Using:
‚Ä¢ Spring Security
‚Ä¢ JWT
‚Ä¢ OAuth2

18. What is Spring Boot CLI?

Command-line tool to create apps using Groovy.

19. Difference between JAR and WAR in Spring Boot?

‚Ä¢ JAR: standalone app with embedded server (preferred)
‚Ä¢ WAR: deployed on external server

20. What is @EnableAutoConfiguration?

Tells Spring Boot to configure beans automatically.

21. What is Actuator Health Check?

Provides application health info using /actuator/health.

22. How to enable Swagger in Spring Boot?

Add dependency ‚Üí Annotate with @OpenAPIDefinition.

23. What is spring-boot-starter-web?

Starter for building REST APIs (Spring MVC + Tomcat).

24. What is spring-boot-starter-data-jpa?

Starter for JPA + Hibernate + Spring Data.

25. How to connect Spring Boot to MySQL?

spring.datasource.username=root
spring.datasource.password=1234

26. What is CommandLineRunner?

Runs code on startup.

27. Difference between @Bean and @Component?

‚Ä¢ @Component ‚Üí automatic detection
‚Ä¢ @Bean ‚Üí manual bean definition inside config class

28. What is @EnableScheduling?

Used for scheduled tasks (cron jobs).

29. How to create cron job in Spring Boot?

30. How to profile environments?

Use @Profile:
