Array Data Structure - Complete Guide

üöÄ What is an Array?

An array is a collection of elements stored at contiguous memory locations. It allows random access to elements using indices (0-based in most languages).

üìä Key Characteristics

  • Fixed Size: Size is determined at declaration
  • Homogeneous: All elements are of the same type
  • Random Access: Access any element in O(1) time
  • Contiguous Memory: Elements are stored consecutively
  • Zero-Indexed: First element at index 0

‚è±Ô∏è Time Complexity

  • Access: O(1) - Direct index access
  • Search: O(n) - Linear search, O(log n) - Binary search (sorted)
  • Insertion: O(n) - Need to shift elements
  • Deletion: O(n) - Need to shift elements

// Reverse
for (int i = 0; i < arr.length / 2; i++) {
    int temp = arr[i];
    arr[i] = arr[arr.length - 1 - i];
    arr[arr.length - 1 - i] = temp;
}

üíª Example: Two Sum Problem

Solution Implementation


// Find two numbers that sum to target
public static int[] twoSum(int[] nums, int target) {
    Map map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (map.containsKey(complement)) {
            return new int[]{map.get(complement), i};
        }
        map.put(nums[i], i);
    }
    return new int[]{};
}

üìã Summary

Arrays are fundamental data structures offering O(1) access time but with fixed size limitations. Perfect for scenarios requiring fast random access to elements.
