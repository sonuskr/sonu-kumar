String Data Structure - Complete Guide

üöÄ What is a String?

A string is a sequence of characters. In Java, strings are immutable objects, meaning once created, their values cannot be changed.

üìä Key Characteristics

  • Immutable: Cannot be modified after creation
  • Sequence of Characters: Ordered collection of chars
  • Zero-Indexed: First character at index 0
  • Comparable: Can be compared lexicographically
  • Unicode Support: Supports all Unicode characters

üîß String Creation

Different Ways to Create Strings


// String literal (stored in string pool)
String s1 = "Hello";

// Using new keyword (stored in heap)
String s2 = new String("Hello");

// From character array
char[] chars = {'H', 'e', 'l', 'l', 'o'};
String s3 = new String(chars);

// Note: s1 == s2 returns false (different memory locations)
// But s1.equals(s2) returns true (same content)

‚ö° Common String Methods

String Operations


String str = "Hello World";

// Length
int length = str.length();  // 11

// Character at index
char ch = str.charAt(0);  // 'H'

// Substring
String sub = str.substring(0, 5);  // "Hello"

// Index of
int index = str.indexOf("World");  // 6

// Case conversion
String upper = str.toUpperCase();    // "HELLO WORLD"
String lower = str.toLowerCase();    // "hello world"

// Contains
boolean contains = str.contains("World");  // true

// Replace
String replaced = str.replace("World", "Java");  // "Hello Java"

// Split
String[] words = str.split(" ");  // ["Hello", "World"]

// Trim
String trimmed = "  Hello  ".trim();  // "Hello"

// Starts/Ends with
boolean starts = str.startsWith("Hello");  // true
boolean ends = str.endsWith("World");      // true

// Equals
boolean equals = str.equals("Hello World");        // true
boolean equalsIgnoreCase = str.equalsIgnoreCase("hello world");  // true

  • Access character: O(1)
  • Search: O(n)
  • Concatenation: O(n)
  • Substring: O(n)

üîÑ String vs StringBuilder vs StringBuffer

Performance Comparison


// String - Immutable
String s = "Hello";
s = s + " World";  // Creates new String object

// StringBuilder - Mutable, NOT synchronized (faster)
StringBuilder sb = new StringBuilder("Hello");
sb.append(" World");
String result = sb.toString();

// StringBuffer - Mutable, synchronized (thread-safe)
StringBuffer sbf = new StringBuffer("Hello");
sbf.append(" World");
String result2 = sbf.toString();

// Use StringBuilder for better performance in loops
StringBuilder sb2 = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb2.append(i);  // Much faster than String concatenation
}

üíª Code Examples

Palindrome Check


public static boolean isPalindrome(String s) {
    // Remove non-alphanumeric and convert to lowercase
    String cleaned = s.replaceAll("[^a-zA-Z0-9]", "").toLowerCase();
    
    // Check from both ends
    int left = 0, right = cleaned.length() - 1;
    while (left < right) {
        if (cleaned.charAt(left) != cleaned.charAt(right)) {
            return false;
        }
        left++;
        right--;
    }
    return true;
}

First Non-Repeating Character


public static char firstNonRepeating(String s) {
    Map charCount = new HashMap<>();
    
    // Count frequency
    for (char c : s.toCharArray()) {
        charCount.put(c, charCount.getOrDefault(c, 0) + 1);
    }
    
    // Find first non-repeating
    for (char c : s.toCharArray()) {
        if (charCount.get(c) == 1) {
            return c;
        }
    }
    return '\0';  // No non-repeating character
}

‚úÖ Best Practices

  • Use String for immutable data
  • Use StringBuilder for string concatenation in loops
  • Use equals() instead of == for string comparison
  • Use equalsIgnoreCase() for case-insensitive comparison
  • Avoid creating unnecessary String objects
  • Use StringBuilder.toString() only at the end

üìã Summary

Strings are immutable sequences of characters in Java. Use StringBuilder for efficient string manipulation and always use equals() for string comparison.
