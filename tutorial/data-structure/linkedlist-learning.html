Linked List - Complete Guide

What is a Linked List?

A linked list is a linear data structure where elements (nodes) are connected via pointers/references. Each node contains data and a reference to the next node.

Types of Linked Lists:

  • Singly Linked List: Each node points to next node only
  • Doubly Linked List: Each node points to both previous and next
  • Circular Linked List: Last node points back to first node

Node Structure:


// Singly Linked List Node
class Node {
    int data;
    Node next;
    
    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

// Doubly Linked List Node
class DNode {
    int data;
    DNode prev;
    DNode next;
    
    DNode(int data) {
        this.data = data;
        this.prev = null;
        this.next = null;
    }
}

Time Complexity:

  • Access: O(n) - Must traverse from head
  • Search: O(n) - Linear search
  • Insertion (at beginning): O(1)
  • Insertion (at end): O(n) without tail, O(1) with tail
  • Deletion (at beginning): O(1)
  • Deletion (at end): O(n)

Basic Operations:


// Traversal
Node current = head;
while (current != null) {
    System.out.println(current.data);
    current = current.next;
}

// Insertion at beginning
Node newNode = new Node(value);
newNode.next = head;
head = newNode;

// Insertion at end
Node current = head;
while (current.next != null) {
    current = current.next;
}
current.next = new Node(value);

// Deletion at beginning
if (head != null) {
    head = head.next;
}

// Deletion at position
if (pos == 0) {
    head = head.next;
} else {
    Node current = head;
    for (int i = 0; i < pos - 1 && current != null; i++) {
        current = current.next;
    }
    if (current != null && current.next != null) {
        current.next = current.next.next;
    }
}

Common Interview Problems:

  • Reverse Linked List: Reverse the entire list
  • Detect Cycle: Check if cycle exists
  • Middle of List: Find middle element
  • Remove Nth Node: Remove node at position n
  • Merge Sorted Lists: Merge two sorted lists
  • Palindrome Check: Check if list is palindrome
  • Intersection Point: Find intersection of two lists

Example: Reverse Linked List


public Node reverseLinkedList(Node head) {
    Node prev = null;
    Node current = head;
    
    while (current != null) {
        Node nextTemp = current.next;  // Store next
        current.next = prev;            // Reverse link
        prev = current;                 // Move prev
        current = nextTemp;             // Move current
    }
    return prev;  // New head
}

Example: Detect Cycle (Floyd's Algorithm)


public boolean hasCycle(Node head) {
    if (head == null) return false;
    
    Node slow = head;
    Node fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;          // Move 1 step
        fast = fast.next.next;     // Move 2 steps
        
        if (slow == fast) {
            return true;  // Cycle detected
        }
    }
    return false;
}

Example: Middle of Linked List


public Node findMiddle(Node head) {
    if (head == null) return null;
    
    Node slow = head;
    Node fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;       // Move 1 step
        fast = fast.next.next;  // Move 2 steps
    }
    return slow;  // Middle node
}

Advantages & Disadvantages:

Advantages:

  • Dynamic size
  • Efficient insertion/deletion at beginning
  • No memory wastage

Disadvantages:

  • No random access - O(n)
  • Extra memory for pointers
  • Cache unfriendly

When to Use:

  • When you need frequent insertions/deletions
  • When size is unknown or varies greatly
  • Implementing stacks and queues
  • Creating graphs using adjacency list
